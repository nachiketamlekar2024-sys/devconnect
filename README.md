# DevConnect

DevConnect is a freelance marketplace that connects clients with skilled developers, providing a platform for job postings, bids, contracts, and secure escrow payments. The platform streamlines the process of finding and hiring top talent, while ensuring timely and secure payments. By leveraging DevConnect, clients can focus on their projects, while developers can focus on delivering high-quality work.

**Target Users:** Freelance developers and businesses looking to hire skilled talent
**Complexity:** High

## Getting Started

### Prerequisites
- Node.js 18+
- MongoDB (or use Docker)

### Quick Start with Docker

```bash
docker-compose up -d
```

### Manual Setup

1. **Install dependencies:**
```bash
cd backend && npm install
cd ../frontend && npm install
```

2. **Set up the database:**
```bash
cd backend
cp .env.example .env
# Edit .env with your DATABASE_URL
```

3. **Start development servers:**
```bash
# From root
npm run dev
```

Backend: http://localhost:3001
Frontend: http://localhost:5173
